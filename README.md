# 🧬 MolSynthAI

**MolSynthAI** is an AI-powered molecular structure generator designed for lead optimization and drug discovery. Built with Streamlit, RDKit, and Google Gemini, it allows medicinal chemists and researchers to input a starting molecule and generate novel, chemically valid structural analogs.

## 🌟 Key Features

* **LLM-Powered Generation**: Uses Google Gemini to act as an expert medicinal chemistry assistant, generating analogs based on logical transformations (e.g., bioisosteric replacements, scaffold hopping, and ring expansions).
* **Robust Chemical Validation**: Ensures generated SMILES strings are chemically valid and adhere to strict, standard atomic valencies.
* **Property Calculation & Profiling**: Automatically computes key physicochemical properties using RDKit for each generated molecule:
  * Molecular Weight (MW)
  * Lipophilicity (LogP)
  * Topological Polar Surface Area (TPSA)
  * H-bond Donors (HBD) & Acceptors (HBA)
  * Rotatable bonds & Ring counts
  * Quantitative Estimate of Drug-likeness (QED) score
* **Lipinski’s Rule of Five**: Provides instant color-coded compliance checks.
* **Tanimoto Similarity**: Calculates Morgan fingerprint similarity against the input molecule.
* **Interactive Visualization**:
  * Adaptive 2D SVG molecular depictions that respond to system dark/light modes.
  * Interactive 3D molecular viewers built with `py3Dmol`.
* **Flexible Exports**: Download generated libraries as:
  * `CSV` summary (contains SMILES and all computed properties)
  * `SDF` (Spatial Data File with 3D coordinates)
  * `SMI` (Standard SMILES format)
* **Automatic API Key Failover**: Supports an array of API keys in configuration, automatically failing over to the next key if a rate limit or quota exhaustion (429) occurs.

## 🛠️ Technology Stack

* **Frontend**: [Streamlit](https://streamlit.io/)
* **Cheminformatics**: [RDKit](https://www.rdkit.org/)
* **Molecular Visualization**: [py3Dmol](https://3dmol.csb.pitt.edu/) / `stmol`
* **AI engine**: Google Gemini (`google-genai`)

## 🚀 Getting Started

### Installation

1. Clone the repository:

```bash
git clone https://github.com/MangalamGSinha/MolSynthAI.git
cd MolSynthAI
```

2. Install the required dependencies (Python 3.10+ recommended):
```bash
pip install -r requirements.txt
```

### Configuration

Create a `.streamlit/secrets.toml` file in the root directory to store your Google Gemini API keys. The app supports a list of keys for automatic failover.

```toml
GEMINI_API_KEYS = [
    "YOUR_GEMINI_API_KEY_1",
    "YOUR_GEMINI_API_KEY_2"
]
```

### Running the application

Launch the Streamlit server:

```bash
streamlit run app.py
```

The app will become available at `http://localhost:8501`.

## ⚠️ Disclaimer

The molecules generated by MolSynthAI are predicted by artificial intelligence and have **not been experimentally validated**. Generated compounds may not be synthesizable, may not exist in nature, and could have unpredictable chemical properties or toxicity profiles. Always verify generated structures through computational validation (e.g., molecular docking, ADMET prediction) and experimental synthesis. **This tool is intended for research and educational purposes only.**
